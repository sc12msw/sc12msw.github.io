Websites!

I have made Amina Sound which is a wordpress multisite based on the flat bootstrap theme for Amina Technologies.

Technologies Used

  • PHP (Wordpress and various plugins)
  • Bootstrap 3.0
  • Jquery (Various plugins)
  • SASS (Using my variation of BEM)
  • Gulp for Pre Processing of SCSS
  • Vargrant for local enviroment to develop (scotchbox)

REST API!

All mobile apps are served by REST appication servers. The iOS server uses xml to parse the data. The Android server uses json. Both have bootstrap based admin screens. Django uses its own template language to compiles the modules of the pages.

Mobile Apps!

Powered by a REST backend these mobile apps connect with the cloud to malnipuate the data they need anywhere in the world. The Android clock in and out app is powered by Django Rest-Framework. The iOS drug/pathogen interation app is powered by the Java Jersey Framework. Both served by Apache servers (iOS app uses Tomcat).

Security

The Defense in depth approach: Prevention: Iptables, firewalls, whitelisting, vpns, compartmentalisation, VMs, policy groups, UAC, Security certificate pinning etc. Detection: Canary tokens, wireshark, heuristic monitoring, sysinternals (proexp.exe), honeypots etc. Recovery: Automated backups, disk images, Sandboxy
